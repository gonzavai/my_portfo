Tic Tac Toe - Android

Esta aplicación de ta-te-ti fue desarrollada trabajando con el IDE de Android Studio, en lenguaje JAVA y con una base de datos no-relacional llamada Firebase. Permite jugar entre 2 usuarios (en el mismo dispositivo) o contra la máquina. La misma ha sido desarrollada de manera responsiva, de forma que sus componentes se adapten a diferentes tipos de pantallas.

Web scrapping financiero (Android)

Ésta aplicación también ha sido desarrollada en Android studio, pero utilizando Kotlin y Java en diferentes secciones del mismo. Consiste en un programa que mediante web-scrapping a través de la herramienta Jsoup, obtiene en tiempo real ciertos datos bursátiles previamente definidos. Nuevamente se trabajó con una base de datos no-relacional como Firebase y fue necesario familiarizarse con el concepto de AsyncTask para su desarrollo.

Adiviname (Android)

Desarrollada en Android Studio, con Java como lenguaje estructural, esta aplicación permite jugar un simple juego donde mediante la deducción propia, basándose en la lista de intentos previos hay que adivinar el número aleatoriamente definido por el sistema. Si bien el juego es algo simple, el objetivo del mismo era poder probar diferentes funcionalidades como ser la Geolocalización. También fue desarrollada con Firebase como base de datos.
